# AI Cover Letter Generator

Generate tailored cover letters instantly using AI. Paste your CV and a job description, and get a professional cover letter in seconds.

## Features
- AI-powered cover letter generation
- Paste your CV and job description as input
- Copy generated letter to clipboard instantly
- Clean, minimal UI

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS
- Google Gemini AI API

## Getting Started

1. Clone the repo
```bash
   git clone https://github.com/Rumaisa10/cover-letter-generator.git
   cd cover-letter-generator
```

2. Install dependencies
```bash
   npm install
```

3. Add environment variables — create a `.env.local` file:

4. Run the dev server
```bash
   npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Environment Variables

| Variable | Description |
|---|---|
| `GEMINI_API_KEY` | Your Google Gemini API key from [aistudio.google.com](https://aistudio.google.com) |
